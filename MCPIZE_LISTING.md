## Long Description

## Overview
Google Search Console MCP Server connects AI clients to Google's official Search Console APIs so they can inspect indexing, query search performance, manage sitemaps, and administer properties from one workflow.

It is built for SEO, growth, content, and engineering teams that want live Search Console data inside MCP-native clients such as Codex and Claude Code.

## What You Can Do
- Connect a Google account with secure OAuth and encrypted token storage.
- Query Search Analytics data for clicks, impressions, CTR, average position, and dimension-based breakdowns.
- Inspect a specific URL to check index status, crawl state, and page-level issues.
- List, fetch, submit, and delete sitemaps for a property.
- List, fetch, add, and delete Search Console properties.
- Use built-in confirmation guards for destructive actions and audit logs for mutations.

## Best Fit
- Technical SEO workflows
- Indexing and crawl diagnostics
- Automated reporting
